# RR Nagar — Backend

This repository contains the RR Nagar backend for the project. It provides the API and server-side logic used by the frontend, including authentication and admin flows.

Key features
- Express-based API server
- Supplier & Customer authentication
- Google OAuth integration
- KYC flow with admin approval
- Deployed on Vercel

Notes
- This README focuses on the backend; see the `src/` and `public/` folders for frontend code and assets.
- For local development, use the provided scripts or `docker-compose.yml` when available.
